# LearnLanguage

**LearnLanguage** to projekt aplikacji do nauki języków obcych. Głównym celem jest stworzenie prostego i skutecznego narzędzia, które ułatwia codzienne przyswajanie słówek i gramatyki. Dzięki technologii .NET MAUI aplikacja ma docelowo działać na wielu systemach, pomagając przełamywać bariery językowe na komputerze i telefonie.
